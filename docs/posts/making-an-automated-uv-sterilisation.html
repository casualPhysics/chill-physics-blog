Making an Automated UV Sterilisation Pump

15 October 2025 · Build log

Thanks to David Jordan at Living Physics for the supervision!

I wanted to make an automated setup to test the effect of UV sterilisation of yeast cells at different doses. I now have the concentration that I need to make a countable colony.

To do this, I have a source tube with the solution that I want to irradiate. A pump, which is connected to a relay, pumps it into a cuvette. The cuvette holder has a window where I can shine a different UV lights into it. The pump then finally moves it into the final tube.

The relay is connected to an Arduino nano so I can control the pump from my mac. I can also connect UV lights to the relay to control the dose from there as well.
